# LangkahKecil Landing Page

Landing page untuk **LangkahKecil** — aplikasi pencatat dan pemantau tumbuh kembang anak.

## ✨ Fitur Landing Page

- 🎨 Desain modern & responsive
- 📱 Mobile-first design
- ⚡ Animasi smooth (scroll reveal, floating elements)
- 🎯 SEO-optimized
- 🚀 Pure HTML/CSS/JS — tanpa framework

## 🚀 Cara Menjalankan

Cukup buka `index.html` di browser, atau gunakan live server:

```bash
# Menggunakan Python
python -m http.server 8000

# Menggunakan Node.js
npx serve .
```
